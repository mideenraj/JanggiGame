----- Description ----- 
This program is the implementation of the two player Japanese chess board game, Janggi. The game is played by two players, BLUE and RED, with BLUE always starting the game off by making the first move. The game ends when a player is checkmated and cannot make any valid moves without losing their most significant piece, the General.


----- How to run the program ----- 
To run the program, simply run the JanggiGame.py file through a command terminal or IDE. The function main, containing the driver code with loop, prompting two players to make turns until the game has ended.


Please try out the game and I hope you enjoy playing!
